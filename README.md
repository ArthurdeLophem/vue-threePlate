# Vue 3 + three
this boilerplate makes it easy to use three in vue by using oop. you'll save plenty of time, say thanks.


<p>&nbsp;</p>

### how to use

install locally
````
git clone
````

explore to the filebase
````
cd vue-threePlate
````

install packages
````
npm i
````

run the thing (explore the possibilities of threes)
````
npm run dev
````
